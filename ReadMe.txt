作品の概要
制作環境　Windows11 DirectX12 C++
制作期間　2022/06-09
制作人数　一人
実行手順　SirakamiKoumakyouのGame2D.exeから実行することが出来ます。
作品概要　東方紅魔郷をホロライブバージョンで再現するような形で作成しました。
　　　　　ゲームの流れや要素・操作方法は原作に準拠しています。

操作方法

タイトル時
	矢印キー　↑↓　で選択移動
	ｚキーで決定
	ｘキー

ゲーム時
	矢印キー　↑↓←→　でプレイヤーの移動
	ｚキー　長押しで攻撃
	ｘキー　でボムの使用です。

自己評価
入学後半年の作品としては、たいへん良く頑張ったと思います。
自身の出来ることだけでゲームを作り成長をおろそかにしない為に、すでに世にある物を再現することを目標に制作を行いました。
その結果、特に継承関係の使用やポインター概念・三角関数の習得に大きく貢献しました。
しかし提出に当たり、再度確認したところ、マジックナンバーの使用やコメントの不足・スクリプトのファイル構成など基礎的な点で至らないところも多かったと痛感しています。
paizaの最近のコードと比較して成長を見ていただけると幸いです。

他者評価
バンタンゲームアカデミーでの最初の審査会で同じ入学年度のなかでプレゼンテーションをした際に最優秀賞をいただく事ができました。
その際には、特に
1.DirectXによるC++での制作である点。
2.三角関数を使ったレーザー等の当たり判定の実施。
3.データの読み込み等の重たい内容をスレッドを使いメインループ外で行ったこと。
を大きく評価していただきました。
逆に、改善点として
1.ゲームとしての独自性
2.演出面
3.スクリプトの可読性
を指摘していただきました。
